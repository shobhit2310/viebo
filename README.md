# Viebo

Viebo is an **event-based dating platform** designed to help people connect with others attending the same real-world events. Instead of endless swiping, Viebo allows users to meet people who are present at the same college fest, party, concert, or social gathering.

The platform focuses on **real-world interactions**, making connections more natural, relevant, and exciting.

---

## Idea

Most dating apps connect strangers randomly online.
Viebo changes this by connecting people **within the same event environment**.

If two people are attending the same event, they already share a common experience, making conversations easier and more meaningful.

---

## Key Features

* **Event-Based Matching**
  Users join an event using a unique event code.

* **Location-Based Access**
  Only users within the event location can participate.

* **Real-Time Connections**
  Discover and connect with people attending the same event.

* **Safer Community**
  Event codes and location verification help prevent outsiders from joining.

* **Designed for Social Events**
  Perfect for college fests, parties, clubs, meetups, and concerts.

---

## Tech Stack

* **Frontend:** React Native + Expo
* **Backend:** Supabase
* **Authentication:** Supabase Auth
* **Database:** PostgreSQL (via Supabase)
* **Location Services:** Map APIs

---

## How It Works

1. User installs the Viebo app.
2. User joins an event using an **event code**.
3. App verifies the **user location** within the event radius.
4. Users attending the event can **discover and interact with each other**.
5. If two users express interest, they can start chatting.

---

## Vision

Viebo aims to bring back the magic of **meeting someone in the moment**.
By combining real-world events with technology, the platform creates opportunities for **authentic connections, friendships, and relationships**.
